How this sample app works

  • This example code was deployed to Heroku.
  • It runs this website and serves up this web application.
  • The webapp can interact with Fincura to submit documents.
  • Displaying data that is contained in those documents as they move through the platform
  • It demo's how third parties can integrate these steps into their workflow.
